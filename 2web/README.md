# Todo App

A simple Todo application built with Spring Boot, JPA, H2, and Thymeleaf.

## Prerequisites

- Java 17 or higher
- Maven 3.6 or higher

## Running the Application

1. Clone or download the project.
2. Navigate to the project directory.
3. Run `mvn spring-boot:run`
4. Open http://localhost:8080 in your browser.

## Features

- Add new todos
- View all todos
- Delete todos
